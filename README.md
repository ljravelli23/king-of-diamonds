# ♦ King of Diamonds ♦

Juego multijugador online inspirado en el "Rey de Diamantes" de **Alice in Borderland**. 
¡Juega 2 jugadores online al instante sin configuración!

## 📋 Descripción

Un juego de estrategia y predicción psicológica. Cada participante elige un número entre 0 y 100. El objetivo es acercarse al **Promedio de todos los números × 0.8**. El más alejado pierde puntos. Si llegas a -10 puntos, estás eliminado.

## 🚀 Características (Nueva Versión)

- **Online Instantáneo**: Usa **PeerJS (P2P)** para jugar online compartiendo un link. Sin necesidad de configurar bases de datos.
- **Estética Alice in Borderland**: Diseño premium, oscuro, con efectos de glitch y neón.
- **Sincronización Total**: Estados de juego, puntuaciones y rondas sincronizadas en tiempo real.
- **Reglas de Supervivencia**: Sistema de puntos (-10 para eliminación).
- **Responsive**: Optimizado para PC y móviles.

## 🎮 Cómo Jugar

1. Abre `index.html`.
2. Ingresa tu nombre.
3. **Crea una sala** y comparte el link o el ID con un amigo.
4. Una vez que ambos estén listos, ¡comienza el juego!
5. Elige tu número estratégicamente. ¿Qué pensará tu rival?

## 🛠️ Tecnologías

- **HTML5 & CSS3**: Diseño premium con Glassmorphism.
- **JavaScript (ES6+)**: Lógica de juego avanzada.
- **PeerJS**: Conectividad P2P online sin servidor.

## 🎯 Reglas

1. Todos eligen un número (0-100).
2. Se calcula: `Objetivo = Promedio * 0.8`.
3. El jugador más alejado del objetivo pierde 1 punto.
4. El juego continúa hasta que alguien llega a -10 puntos.
